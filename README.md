# Cinestream_OTT_Platform

## Overview

The OTT Platform is a cloud-based video streaming web application developed using AWS Cloud services. It allows users to browse movies, watch trailers, and stream videos stored in Amazon S3.

## Features

- Movie listing
- Video streaming
- Movie poster display
- Responsive UI
- Cloud-hosted application
- Stores videos and posters in s3 bucket
- Serverless backend using Lambda

## Technologies Used

- HTML
- CSS
- JavaScript
- Python
- AWS EC2
- AWS S3
- API Gateway
- AWS Lambda
- DynamoDB
- Boto3

## AWS Services

- Amazon EC2
- Amazon S3
- AWS Lambda
- API Gateway
- IAM
